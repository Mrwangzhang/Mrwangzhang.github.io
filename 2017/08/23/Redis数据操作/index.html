Redis数据操作

Author: John Doe Date: August 23, 2017  21:21:14 Category: python

学习目标

  • 了解Redis存放数据结构

数据结构

  • redis是key-value的数据结构，每条数据都是一个键值对
  • 键的类型是字符串
  • 注意：键不能重复
  • 值得类型分为五种：
    • 字符串string
    • 哈希hash
    • 列表list
    • 集合set
      • 不可重复
      • 无序
    • 有序集合zset
      • 有序是按照一定序列排序

数据操作行为

  • 保存
  • 修改
  • 获取
  • 删除
